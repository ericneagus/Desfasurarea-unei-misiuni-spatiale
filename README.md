# Desfasurarea unei misiuni spatiale
Acest proiect reprezinta un simulator complex de zbor spatial care gestioneaza navigarea interlplanetară, integrarea sistemelor critice și supraviețuirea echipajului într-un mediu ostil.

# Descriere generală
Programul simuleaza călătoria unei nave spațiale către o locație fixă (cu coordonate 3d). Pe parcursul zborului, nava este supusa unor hazarde generate aleatoriu care afectează atât nava spațială cât și echipajul acesteia.

# Caracteristici Principale
A) Navigatie vectorială: Nava calculeaza traiectoria directa catre destinatie folosind formule de distanta euclidiana.
B) Gestiune polimorfică: Fiecare componenta reactioneaza diferit la impact prin metode virtuale.
C) Motor de evenimente: Un simulator de hazard ce determina soarta misiunii prin probabilitati statistice.
